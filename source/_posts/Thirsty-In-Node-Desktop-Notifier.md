#### Today I created Thirsty In Node Desktop Notifier.

# Configuration

## (1) Install dependencies

`npm install` or `npm i`

## (2) Add it in PM2

`npm start`

## (3) Start it during PC BOOT

`npm run startup`

## (4) To delete it from PM2

`npm run delete`
